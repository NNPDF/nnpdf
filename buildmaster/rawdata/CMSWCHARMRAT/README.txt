This is the raw data for ratios of W+ and W- cross sections in the
CMS measurement of W+charm production arxiv:1310.1138

We use the data for ptlepton > 25 GeV

The experimental data has been extracted from Table 7 in arxiv:1310.1138.
The recommendation from the analyzers is to treat the data uncorrelated,
both between bins and with the absolute differential W+charm production
data
